# Installation

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  laravel_rest_api_flutter: latest_version
  dio: ^5.0.0 # or your preferred HTTP package
```

Then run:

```bash
flutter pub get
```

You are now ready to configure your HTTP client and repositories.
